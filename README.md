# 📊 Financial Analytics Dashboard | Power BI

## 📌 Visão Geral do Projeto

Este projeto consiste no desenvolvimento de um **Dashboard de Análise Financeira** construído no Power BI, com foco em transformar dados financeiros em insights estratégicos para suporte à tomada de decisão.

O painel foi projetado com abordagem orientada a negócio, priorizando clareza visual, performance analítica e geração de valor por meio de indicadores financeiros relevantes.

---

## 🎯 Problema de Negócio

Empresas e gestores financeiros precisam acompanhar receitas, despesas e saldo de forma estruturada para:

- Identificar padrões de gastos
- Controlar fluxo de caixa
- Avaliar crescimento financeiro
- Apoiar decisões estratégicas

Este dashboard centraliza essas informações em um único painel interativo.

---

## 📊 Principais Métricas Desenvolvidas

- 💰 Receita Total
- 💸 Despesa Total
- 📈 Saldo Consolidado
- 📅 Análise Temporal (mensal e anual)
- 📂 Distribuição por Categoria
- 📊 Tendência e evolução financeira
- 📉 Comparativos entre períodos

---

## 🧠 Técnicas Aplicadas

- ✔ Modelagem relacional de dados
- ✔ Criação de medidas com DAX
- ✔ Tratamento e organização de dados
- ✔ Construção de KPIs financeiros
- ✔ Design focado em UX e Data Storytelling
- ✔ Estruturação de dashboard orientado à decisão
